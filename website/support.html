Support

Help for PromoKit: Offer Codes.

Use these resources for setup questions, bug reports, and feature requests.

Read the Project Guide

The repository README explains local seed data, CSV import expectations, saved state, and development setup.

Report an Issue

Open a GitHub issue at github.com/Erik-Handeland/PromoKit/issues. Include your iOS version, app version, and the steps that reproduced the problem.

Code Import Help

PromoKit expects App Store Connect offer-code CSVs with code and redeem URL columns. Real promo-code CSVs should stay private and should not be committed to the repository.

App Review Note

The app can be reviewed with the bundled demo data. No demo account is required.

PromoKit is free, open source, and built for App Store developers.
